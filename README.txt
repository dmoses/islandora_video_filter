
========= Installation =========

Enable the module on the modules page.  You will also need the Video Filter module
enabled.  You will also need to properly configure the text format(s). See the
Video Filters README for further instructions.

========= Troubleshooting =========

If videos don't show up, try disabling any browser plugins, such as AdBlock.

If you are having issues in a dev env. using localhost you may want to try 127.0.0.1.
The Video Filter module looks for a . in the host name.
